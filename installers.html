FOXMINER Installers

Download the right Foxminer version for your system below!

macOS ARM64 (AArch64)

For Apple Silicon Macs (M1, M2, etc.). Optimized for ARM architecture.

Open-source, not from the App Store. After downloading:
1. Open Foxminer-v1.0.dmg
2. Drag Foxminer.app to Applications
3. Go to System Settings > Security & Privacy > "Open Anyway" to allow it.

Intel x86_64 (10.9.5 and newer)

For Intel-based Macs running macOS 10.9.5 or later.

Windows ARM

For Windows devices with ARM processors (e.g., Surface Pro X).

Windows x86-64

For 64-bit Windows systems (most modern PCs).

Linux

For Linux distributions (64-bit). Works on Ubuntu, Debian, etc.
