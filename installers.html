FOXMINER Installers

Download the right Foxminer version for your system below!

macOS ARM64 (AArch64)

For Apple Silicon Macs (M1, M2, etc.). Optimized for ARM architecture.

Linux ARM64 (AArch64)

For Linux distributions on ARM processors (e.g., Raspberry Pi, ARM servers).

How to Install

macOS ARM64

  1. Download foxminer-v1-macos.dmg using the button above.
  2. Double-click the DMG to mount it, or use: hdiutil attach foxminer-v1-macos.dmg
  3. Open Terminal and navigate to the mounted volume: cd /Volumes/Foxminer
  4. Run the miner: ./foxminer -c config.json
  5. Enter your Monero wallet address when prompted. It will be saved to /Volumes/Foxminer/config.json.
  6. To run again, remount the DMG and repeat steps 3-4. No need to re-enter your wallet.

Linux ARM64

  1. Download foxminer-v1-linux.tar.gz using the button above.
  2. Extract the tarball: tar -xvzf foxminer-v1-linux.tar.gz
  3. Navigate to the extracted directory: cd [extracted-folder]
  4. Run the miner: ./foxminer -c config.json
  5. Enter your Monero wallet address when prompted. It will be saved to config.json.

Note: Foxminer is open-source and not from the App Store or package managers. Keep the macOS DMG file, as the wallet is saved in /Volumes/Foxminer/config.json when mounted.
